== Description ==

Allows Wordpress Admin to send SMS Reminder messages to users who sign up for text reminders. New Users are signed up at register but can opt out by erasing and saving the phone number field blank.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "Twilio Text SMS".
3. Click install.
4. Click activate.
